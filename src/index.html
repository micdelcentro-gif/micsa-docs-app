Google Docs

Integración Google Docs API

Necesitas un proyecto en console.cloud.google.com

💡

¿Cómo configurar?

1. Ve a console.cloud.google.com
2. Crea un proyecto → APIs & Services → Habilita Drive API + Docs API
3. Crea credenciales: OAuth 2.0 Client ID (tipo Web) + API Key
4. Añade file:// a los orígenes autorizados
