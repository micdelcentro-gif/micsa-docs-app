# Stack React
- Componentes funcionales con hooks únicamente — sin clases
- Estado local: useState/useReducer; estado global: context o zustand
- Efectos: useEffect con cleanup explícito
- No usar index como key en listas dinámicas
- Separar lógica en custom hooks reutilizables
